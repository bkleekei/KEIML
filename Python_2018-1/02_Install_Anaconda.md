### 아나콘다 설치하기  
  
  1) 보통 다운로드 폴더에 설치파일이 받아집니다.  
  ![설치파일](./imgs/installfile.png)  
  2) 더블클릭 등으로 설치를 시작합니다.  
  Next를 클릭합니다.  
  ![설치](./imgs/install1.png)  
  3) 사용권 게시로 Next를 클릭합니다.  
  ![설치](./imgs/install2.png)

  ![설치](./imgs/install3.png)
